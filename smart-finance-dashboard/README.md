# Smart Finance Dashboard

A real-time React financial management tool.

## Features
- **CRUD Operations**: Add, Edit, and Delete transactions.
- **Real-time Analytics**: Monthly savings % and category breakdowns.
- **Optimized Performance**: Uses `useMemo` for heavy calculations.
- **Budget Alerts**: Visual warnings when spending exceeds limits.

## Tech Stack
- React (Vite)
- CSS3 (Responsive Design)
- Lucide React (Icons)

## Setup
1. `npm install`
2. `npm run dev`
